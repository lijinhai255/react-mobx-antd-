# react react-router-dom mobx antd-mobile

react项目的地基项目，已经配置了 react-router-dom 和 mobx

run

```
yarn & yarn start

```

目录结构示意图

```
src     需要打包的所有代码目录
|
|
|---- component  组件目录 
|    
|---- routes  各个路由页面 
|
|---- http    接口目录
|
|---- image   图片目录
|
|---- store   mobx 仓库
|
|---- utils   工具类
|
|---- app.js   路由
|
|---- index.js  入口文件

```
